                     TODO Web App

Веб-приложение для управления списком задач. Написано на Go, данные хранятся в PostgreSQL. Проект запускается в контейнерах Docker.

Технологии:
- Go (Golang)
- PostgreSQL
- Docker / Docker Compose
- HTML

 Инструкция по запуску.

Для запуска рекомендуется использовать Docker, так как он автоматически настраивает базу данных.

1. Клонировать репозиторий:
   git clone https://github.com/annn156/todo-web-app-go.git

2. Перейти в папку проекта:
   cd todo-web-app-go

3. Запустить приложение:
   docker-compose up --build

После запуска приложение будет доступно в браузере по адресу: http://localhost:8080

 Структура проекта:

- internal/ - Логика работы с базой данных
- main.go - Основной файл приложения (HTTP сервер)
- Dockerfile - Настройки сборки приложения
- docker-compose.yml - Настройки запуска контейнеров (App + DB)
- index.html - Шаблон веб-страницы
